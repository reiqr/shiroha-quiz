# Contributing

感谢你关注 `Shiroha Quiz`。

当前项目仍处于早期阶段，欢迎通过 Issue 或 Pull Request 参与改进。

## 可以帮助的方向

- Bug 反馈
- 题库导入与解析兼容性问题
- Web 端 UI / 交互优化
- Android 原生体验优化
- 文档与示例题库补充

## 提交前建议

在提交 Pull Request 前，建议先通过 Issue 说明：

- 你准备解决什么问题
- 改动主要影响 `Web`、`Android` 还是共享逻辑
- 是否会影响题库解析、导入结果或已有数据结构

## 当前优先方向

目前更欢迎这些方向的改进：

- 原生解析器样例与真实题库回归
- 导入格式文档完善
- Compose UI 体验优化
- 备份、恢复、导出能力增强

## 代码与文档约定

- 默认统一项目名称为 `Shiroha Quiz`
- 尽量保持目录边界清晰
- 不要把平台专属逻辑随意混入共享层
- 涉及导入解析逻辑时，优先补样例和验证说明
- 涉及公开上传时，避免引入任何敏感、专有或个人信息
