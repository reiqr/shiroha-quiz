# Shiroha Quiz 原生 Compose 开发进度

> 当前状态更新：原生 Compose 版已进入主要开发线，练习、考试、错题、收藏、智能复习、导入解析、AI 辅助和备份互通均已落地。

## 项目概述

Shiroha Quiz 目前同时维护 WebView 壳版和原生 Compose 版：

| 入口 | 技术栈 | 包名 | 状态 |
|---|---|---|---|
| WebView 壳版 | WebView 加载本地 HTML/CSS/JS | `com.yiqiu.shirohaquiz` | 功能完整，进入稳定维护 |
| 原生 Compose 版 | Kotlin + Jetpack Compose + Material3 | `com.reqir.shirohaquiz` | 当前主开发线，持续完善体验与解析稳定性 |

当前原生主线重点是：稳定导入解析、优化刷题体验、完善错题复习和学习记录，并继续收口小屏与异常显示设备兼容。

## 已完成

- 原生首页、底部导航、题库概览与快捷入口。
- 题库导入、解析预览、导入状态展示和异常提示。
- 标准题库解析、答案区解析、双文件合并、表格导入、策略评分与异常分级。
- JSON 导入全面兼容：多题库数组、题目数组、单题库对象、Web 备份包、key-value 对象选项。
- 导入解析器重构：选项检测逻辑集中管理，修复英文单词内字母误判为选项标记。
- 题库列表、题库详情、切换、删除、重命名、题目预览与二次核对。
- 练习模式：顺序/随机、批量练习、即时反馈、自动下一题、背题模式、斩题功能。
- 练习增强：题目快速编辑、题目收藏、字号缩放、紧凑选项、顺序练习进度记忆、退出保存当前位置。
- 选项打乱（练习+考试独立控制）：固定 A/B/C/D 标签，仅随机内容映射，本次练习/考试内顺序不变。
- 考试模式：题量设置、倒计时、答题卡、自动交卷、得分统计。
- 错题本：错题收录、掌握状态、连续答对规则、智能复习和复习数量提示。
- 错题本作用域切换：按当前题库或全部题库筛选错题列表、首页统计和复习范围。
- 刷题记录：记录列表与详情支持只看错题筛选。
- AI 核对 / AI 解析 / AI 单题分析、编辑器 AI 补解析组件。
- 数据导入导出、JSON / ZIP 备份、Web 与原生题库互通。
- 素材插画接入、Shiroha 模式、暗色主题、开屏图和基础视觉规范。
- 小屏 / 显示异常设备兼容：首页紧凑布局阈值调整，题目文本允许自动换行。
- Question 云端字段完整序列化（subject、grade、difficulty 等 11 字段）。
- 回归测试 20 用例积累，17/20 通过。

## 核心模块

- `QuizRepository`：题库、练习、考试、错题、收藏、智能复习、记录的集中状态管理。
- `importer/`：文本标准化、题目解析、答案解析、双文件合并、表格导入、策略评分与校验。
- `ai/`：AI 核对、AI 解析生成、OpenAI 兼容 API。
- `ui/components/`：GlassCard、ActionPillButton、QuizOptionCard、NoticeCard、StatusChip 等组件。
- `ui/theme/`：Material3 主题、间距令牌、字体和颜色系统。
