一、代码表达式防误识别
1. 阅读代码，判断最终输出。
ls = []
ls.append(Da)
text = "(AD)"
if (A) {
    run()
}
// 注释中的 (B) 不是答案
# 注释中的 (C) 也不是答案
A. 输出一
B. 输出二
C. 输出三
D. 输出四

2. 阅读以下伪代码并说明作用。
A.append(value)
B.remove(value)
C.clear()
答案：依次调用三个方法

3. 阅读字段访问表达式并说明含义。
A.id, B.name, C.value
答案：读取三个对象字段

4. 计算表达式的值。
1.0 + 2.0
答案：3.0
