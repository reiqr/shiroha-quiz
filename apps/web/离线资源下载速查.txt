离线库下载脚本

以下功能默认使用 CDN 加载，如需离线使用请运行对应脚本（在libs文件中）：

  download-pdfjs.ps1     → 下载 PDF.js 5.7.284 完整包（含 CMaps + 字体 + ICC + WASM 图片解码资源）
  download-mathjax.ps1   → 下载 MathJax 3.2.2（约 24 MB）
  ocr/download-ocr.ps1   → 下载 Tesseract.js 5.1.1 完整 OCR 引擎 + 中/英语言包（约 59 MB）

不运行脚本也可正常使用——CDN 会自动兜底。

PDF.js 加载顺序：本地 libs/pdf.min.mjs → CDN jsDelivr → CDN unpkg → 内置轻量提取器
PDF.js 本地增强资源：libs/cmaps/、libs/standard_fonts/、libs/iccs/、libs/wasm/、libs/image_decoders/
MathJax 加载顺序：本地 libs/mathjax/ → CDN jsDelivr → 降级文本显示

OCR 扫描件兜底：文字层 PDF 提取失败或文字过少时自动显示 OCR 面板，详见 libs/ocr/download-ocr.ps1 和 Web 版导入页。
